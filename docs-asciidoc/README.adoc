== About This Documentation

This directory contains AsciiDoc (`.adoc`) versions of the rusty-comms documentation,
suitable for generating PDF and other enterprise documentation formats.

== Building the Documentation

=== Prerequisites

Install Asciidoctor:

[source,bash]
----
# Ruby gem (recommended)
gem install asciidoctor asciidoctor-pdf

# Or via package manager
# Fedora/RHEL
sudo dnf install asciidoctor rubygem-asciidoctor-pdf

# Ubuntu/Debian
sudo apt install asciidoctor
----

=== Generate HTML

[source,bash]
----
asciidoctor *.adoc -D output/
----

=== Generate PDF

[source,bash]
----
asciidoctor-pdf user-guide.adoc -o rusty-comms-user-guide.pdf
----

== Converting from Markdown

To convert Markdown files to AsciiDoc:

[source,bash]
----
# Using pandoc
pandoc -f markdown -t asciidoc ../docs/user-guide/getting-started.md -o getting-started.adoc

# Or using kramdoc (Ruby)
gem install kramdown-asciidoc
kramdoc ../docs/user-guide/getting-started.md -o getting-started.adoc
----
